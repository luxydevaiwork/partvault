Informativa sulla Privacy (Privacy Policy)

1. Titolare del Trattamento

PartVault è un'applicazione sviluppata da un singolo sviluppatore indipendente. Per qualsiasi richiesta relativa alla privacy, puoi contattarci all'indirizzo: [la tua email].

2. Dati Raccolti

L'app non raccoglie, trasmette né condivide alcun dato personale. Tutti i dati inseriti (oggetti, categorie, immagini) vengono salvati esclusivamente sul dispositivo dell'utente.

3. Archiviazione Locale

I dati sono memorizzati in un database SQLite locale e in file immagine nella directory privata dell'app. Nessun dato viene inviato a server esterni o servizi cloud.

4. Permessi Richiesti

L'app può richiedere i seguenti permessi di sistema, utilizzati esclusivamente per le funzionalità descritte:

  • Fotocamera / Galleria: per aggiungere immagini agli oggetti inventariati.
  • NFC: per leggere e scrivere tag NFC associati agli oggetti.
  • Notifiche: per inviare avvisi di scadenza e promemoria di manutenzione.
  • Biometria / PIN: per il blocco opzionale dell'app (i dati biometrici non vengono mai letti né salvati dall'app).

Nessuno di questi permessi viene utilizzato per raccogliere dati sull'utente.

5. Backup ed Esportazione

La funzione di esportazione (JSON/CSV) crea file sul dispositivo dell'utente. La gestione, condivisione e conservazione di tali file è sotto la piena responsabilità dell'utente.

6. Servizi di Terze Parti

L'app non integra analytics, pubblicità, tracciamento o SDK di terze parti di alcun tipo.

7. Minori

L'app non raccoglie dati e non è rivolta specificamente ai minori. Non vi sono rischi particolari per questa categoria di utenti.

8. Modifiche alla Privacy Policy

Eventuali aggiornamenti a questa informativa saranno notificati tramite aggiornamento dell'app sul Google Play Store. La versione più recente sarà sempre disponibile a questo indirizzo.

9. Contatti

Per domande sulla privacy: [la tua email].
